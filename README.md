# Text Analysis — Консольный текстовый анализатор

Простое консольное приложение на C# для анализа текстовых данных. Загружает текст из файла или ввода с клавиатуры, очищает его от лишних символов, подсчитывает частоту слов и формирует топ-10 наиболее употребляемых слов. На приложении практиковались навыки тестирования.

## 🔍 Основные возможности
- Очистка текста: остаются только буквы, апострофы/дефисы **внутри** слов и одиночные пробелы  
- Поддержка латиницы и кириллицы  
- Четыре алгоритма сортировки: встроенная, пузырьком, Шелла, слиянием  
- Сохранение результатов в JSON-файл  
- Расчёт статистики: общее количество слов, уникальных слов, процент уникальности  

## 🧪 Тестирование
- Модульные тесты с использованием xUnit  
- Применены 4 техники проектирования тестов: BVA, ECP, Branch Coverage, Statement Coverage  
- Параметризованные тесты, предположения, мокирование  
- Качество тестов оценено с помощью мутационного тестирования (Stryker.NET)  

## 🛠️ Технологии
- C# / .NET 8  
- xUnit + Moq + Xunit.Assume  
- Stryker.NET для мутационного тестирования  
- Visual Studio 2026
